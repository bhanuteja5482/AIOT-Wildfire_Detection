# AIoT Wildfire Detection using YOLOv8 🔥

This project detects wildfire, fire and smoke using Artificial Intelligence.

## Features
• Detect fire in images  
• Detect smoke in images  
• Batch image detection  
• Automatic result saving  

## Technology Used
• Python  
• YOLOv8 (Ultralytics)  
• Computer Vision  
• AIoT  

## How to Run
Install packages:
pip install ultralytics opencv-python

Run detection:
python test_yolo.py

Output will be saved inside:
runs/detect/predict
